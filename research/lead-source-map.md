# Lead Source Map

Date: 2026-06-24

## Best source types

### Official Reboot Imagine ecosystem pages

Use for calibration of partners, ICPs, and service positioning.

Sources:

```text
https://www.rebootimagine.com/
https://www.rebootimagine.com/it-support
https://www.rebootimagine.com/softwarevendor-services
https://www.rebootimagine.com/about-us
```

### Official organization websites

Use to verify:

```text
current organization existence
current program status
leadership/contact pages
public contact routes
current XR program pages
```

### Public news articles

Use to identify:

```text
funding
large deployments
partnerships
enterprise/customer scale
recent program launches
```

Examples used:

```text
https://www.axios.com/2022/03/24/osso-vr-nets-66m-series-c
https://www.axios.com/2021/09/29/virtual-reality-training-walmart-nfl
```

### Public profile pages

Use only as a seed or weak confirmation source.

Examples used:

```text
https://en.wikipedia.org/wiki/Derek_Belch
https://en.wikipedia.org/wiki/Virti
https://en.wikipedia.org/wiki/VirtaMed
https://en.wikipedia.org/wiki/Amitai_Ziv
https://en.wikipedia.org/wiki/Virtual_Reality_and_Education_Laboratory
https://en.wikipedia.org/wiki/J_Dakota_Powell
https://en.wikipedia.org/wiki/Guy_Primus
```

Risk:

```text
May be stale.
Must be verified against official current sources before outreach.
```

## Source quality rules

High quality:

```text
official organization page
official press release
credible news article
public procurement page
conference page
partner/customer page
```

Medium quality:

```text
public profile
industry blog
conference recap
```

Low quality:

```text
AI-generated listicle
unverified directory
consumer review page
social-only mention
```
